# Match Data Schema (Lunascouter V2)

This document defines the structure and possible values for match scouting data generated by the application.

## Root Object

| Field | Type | Description |
| :--- | :--- | :--- |
| `team` | `string` | Team number (e.g. "254") |
| `matchNumber` | `number` | Current match number |
| `scoutName` | `string` | Name of the scout |
| `values` | `object` | Container for all match metrics |

## Values Object (`values`)

### Autonomous Period
| Field | Type | Possible Values |
| :--- | :--- | :--- |
| `autoPosition` | `number` | `0` (None), `1`, `2`, `3`, `4` |
| `autoFuelCollected`| `string` | `"Center Area"`, `"Human Player"`, `"Depot"`, `"None"` |
| `autoLevel` | `number` | `0` (None), `1` (L1) |
| `movedFromStart` | `boolean` | `true` (Moved), `false` (Didn't Move) |
| `autoScoredZeroFuel`| `boolean` | `true` (Zero Fuel), `false` (Scored Fuel) |

### Teleoperated Period
| Field | Type | Possible Values |
| :--- | :--- | :--- |
| `teleopScoredZeroFuel`| `boolean`| `true` (Scored Fuel), `false` (Zero Fuel) |
| `defense` | `boolean` | `true` (Played Defense), `false` (No Defense) |
| `relayedFuel` | `boolean` | `true` (Transported Fuel), `false` (No Fuel Relay) |
| `mobilityIssues` | `number` | Count (0, 1, 2, ...) |
| `fieldCrossing` | `string` | `"Trench"`, `"Bump"`, `"Both"`, `"None"` |

### Endgame
| Field | Type | Possible Values |
| :--- | :--- | :--- |
| `teleopLevel` | `number` | `0` (None), `1`, `2`, `3` |
| `endgameScoredZeroFuel`| `boolean`| `true` (Zero Fuel), `false` (Scored Fuel) |
| `teleopNote` | `string` | Text comments |

### Post-Match / Flags
| Field | Type | Possible Values |
| :--- | :--- | :--- |
| `brokeDown` | `boolean` | `true` (Broke Down), `false` (No Issues) |
| `needsAttention` | `boolean` | `true` (Needs Review), `false` (Good Scouting) |

---

## Example JSON Blob

```jsonc
{
  "team": "254", // Team number (string)
  "matchNumber": 42, // Current match number
  "scoutName": "John Doe", // Name of the scout
  "values": {
    // --- Auto Metrics ---
    "autoPosition": 1, // 0=None, 1-4=Field zones
    "autoFuelCollected": "Center Area", // "Center Area", "Human Player", "Depot", "None"
    "autoLevel": 1, // 0=None, 1=L1
    "movedFromStart": true, // Whether robot left starting zone
    "autoScoredZeroFuel": true, // true=Zero Fuel (Default), false=Scored Fuel

    // --- Teleop Metrics ---
    "teleopScoredZeroFuel": false, // true=Scored Fuel, false=Zero Fuel (Default)
    "defense": false, // Whether robot played defense
    "relayedFuel": true, // Whether robot transported fuel across field
    "mobilityIssues": 0, // Count of movement/stalling issues
    "fieldCrossing": "Trench", // "Trench", "Bump", "Both", "None"

    // --- Endgame Metrics ---
    "teleopLevel": 3, // 0=None, 1=L1, 2=L2, 3=L3
    "endgameScoredZeroFuel": false, // true=Zero Fuel (Default), false=Scored Fuel
    "teleopNote": "Great robot performance!", // Scout's text comments

    // --- Status Flags ---
    "brokeDown": false, // Mechanical failure flag
    "needsAttention": false // Manual review flag
  }
}
```
